SEGURIDAD INFORMÁTICA

Ciberseguridad o seguridad de tecnología de la información, es el área relacionada con la informática y la telemática que se enfoca en la protección de la infraestructura computacional y todo lo relacionado con esta y, especialmente, la información contenida en una computadora o circulante a través de las redes de computadoras..

INTRODUCCIÓN

OBJETIVOS DEL OVI

• Que el estudiante se fundamente teóricamente sobre las generalidades básicas del Diseño Web.
• Que el estudiante pueda utilizar de forma correcta los lenguajes HTML5 y CSS3 para
aplicarlos en la solución de problemas de construcción de Sitios Web.
• Que el estudiante implemente servicios y sitios web a través de sistemas de control de
versiones.

OBJETIVOS DEL CURSO

• El estudiante podrá desarrollar Sitios Web que satisfagan las necesidades del usuario.
• El estudiante podrá operar lenguajes de Diseño web como HTML5 y CSS3.
• El estudiante podrá operar y administrar sistemas de control de versiones como GIT o
GITHUB.
• El estudiante podrá evaluar Sitios Web y dar recomendaciones concretas para su
actualización.

TEMATICAS DEL CURSO

•Introducción a HTML5.
• Estructura básica de HTML5.
• Aplicación de elementos de HTML5 a un proyecto de diseño Web.
